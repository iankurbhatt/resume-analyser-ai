# AI Resume Analyzer

## What it does
- Extracts text from resume PDF
- Detects skills using NLP
- Compares resume with job description
- Generates match score
- Shows missing skills and recommendations

## Tech Stack
- Python
- scikit-learn
- NLP (TF-IDF)
- Google Colab

## Features
- Resume parsing
- Skill extraction
- Job matching engine
- AI-generated report

## Output Example
Match Score: 72%
Missing Skills: Docker, AWS
